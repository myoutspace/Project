Getting Started

Welcome

Are you ready to post a Task?

Task Manager is ready to answer that question. Task Manager helps you to post and complete tasks within a group of users. Here is what you can do in the app:

  • Create one or more groups
  • Create users in a group
  • Create a task for the group
  • Complete a task for the group

Task Manager allows you to see the statistics of groups, users and tasks easily for better understing of how the group operates

Usage

Task Manager can be used for:

  • Organisations
  • Families
  • Roomates
  • School projects
  • etc ...

Features

  • Facility to post a task
  • Detailed and many options
  • Comprehensive options to customize animation, colors, orientation and style
  • All the power and flexibility of the database
  • Easy install; 100% integration
  • Facility to customize add an icon for a user or group
